Where to learn?

  1. Book: The Way to Go by Karl Baker A small 60 page booklet aimed at total beginners. It will get you familiar with basic rules and some proverbs to keep in mind while playing.

  2. Interctive: OGS's Tutorial

  3. Interactive: Go-Magic / Go Problems

  4. Interactive: "Stone Development for Beginners" by mark5000

  5. Wiki: Sensei's Library Wikipedia of the Go world.

Where to play?

  1. Online-Go-Server (OGS) Live and correspondence games. This is the most popular site for English speaking world. This is where you'll be playing most of your games. This is like lichess of the Go world.

  2. Dragon Go Server Only correspondence games.

  3. BadukPop (Androi/iOS/Web) Mobile app with both live and correspondence games.

Tsumego (Go Puzzles)

  1. GoProblems
  2. Black-To-Play
  3. Tsumego-Hero

All the resources I've mentioned here are legally free to use. I've intentionally kept the list small, so that it doesn't overwhelm a beginner. For more resources refer to Soumya's list.
